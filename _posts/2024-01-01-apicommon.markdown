# APICommon API NuGet Package

A .NET library for easy integration with any public API. This library provides base implementation for intergrating a nuget package with an api.

# What is this currently used in
[PoliceApi.Client](https://www.nuget.org/packages/PoliceApi.Client/)
[RickNMorty.Common](https://www.nuget.org/packages/RickNMorty.Common)

# Installation

You can install the package via NuGet Package Manager or by using the .NET CLI.

```bash
nuget install ApiUtilities.Common
```
or

```bash
dotnet add package ApiUtilities.Common
```

# Usage
## Installation
First, install the NuGet package in your project.

## Register Services
In your application's startup or configuration class, register the services provided by the ApiUtilities.Common library.
